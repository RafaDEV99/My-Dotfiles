# How to include this on your machine:

```

```

# Install deps:

Homebrew (for macOS):
```
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

Nvim (macOS):
```
brew install nvim
```
Zsh (macOS);
```
brew install zsh
```
